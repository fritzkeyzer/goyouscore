# goyouscore

A Go client library for the [YouScore API](https://youscore.com.ua/), primarily generated from their [Swagger documentation](https://api.youscore.com.ua/swagger/index.html#/).

## About

This is a community-maintained library with no affiliation to YouScore.

## Versioning

This project follows [Semantic Versioning](https://semver.org/).
